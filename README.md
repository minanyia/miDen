Proyecto realizado por: Garcia Gaona Romina y Karen Cruces Sánchez 
Descripción del Proyecto
MiDen es una aplicación móvil Android desarrollada en Kotlin con Jetpack Compose que funciona como un organizador personal para gestionar notas y tareas de manera eficiente.

Propósito
La aplicación permite a los usuarios mantener organizadas sus ideas y pendientes en un solo lugar, ofreciendo una interfaz intuitiva y moderna para:

Capturar pensamientos e información importante mediante notas
Gestionar tareas con fechas y estado de completitud
Acceder rápidamente a su información personal


Características Principales
Gestión de Notas

Crear notas con título y descripción
Visualizar todas las notas en lista
Editar notas existentes
Eliminar notas que ya no se necesiten
Persistencia de datos con Room Database

Gestión de Tareas

Crear tareas con título y fecha
Marcar tareas como completadas/pendientes
Editar información de tareas
Eliminar tareas finalizadas
Visualización clara del estado de cada tarea
